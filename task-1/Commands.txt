
# Commands learned
## git init
Initialized a Repository in a local working directory

## git add 
This command will help to stage the changes and make ready for commit (add . -> includes all the current working files, add file_name -> particular file)

## git commit -m branch_name
This command will help to commit in the head point

## git push origin main
This command will help to push the commited changes to the remote repository which is located in github

## git branch brach_name
This helps to create a new branch and every new branch is a isolated environment, by using Branch can able to add a new features for the applications

## git checkout branch_name
This helps to change the head pointer which lead to the pointer of tree structure, when we use (git checkout -b branch_name -> new branch is created and automatically it switches to that branch)
